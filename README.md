# 🌌 Forsaken-Realm

Welcome to **Forsaken-Realm**, a centralized, open-source repository for tabletop campaign lore, world-building data, and historical records. This directory is systematically structured to be highly readable for human players and optimized for local Retrieval-Augmented Generation (RAG) engines.
